• CSS Grids
    • Learned to use grids to break apart sections of a website in a more readable fashion
    • Learned to use semantic tags embedded with classes to order them into box grids with CSS
  • Semantic Tags
    • Tags that dont apply styles but allow applications like web browsers to understand what kind of content they are looking at
    • allows you to make css style changes to many different parts of your website at once
  • Img Tags
    • Allows you to use images on your webpage
    • Has class properties that let you change width height and alternative images for the image
